In this simple code I used:
 * MPEG Algorithm: which is an algorithm is used for encoding videos, movies/motion pictures 

Used Functions:
     * MoviePy: A library for video editing in Python. Used to load, resize and save videos.
     * VideoFileClip(): Loads a video from a file or URL 
     * .resize(): Resizes a video to a new resolution 
     * .size: Gets the resolution (width, height) of a video 
     * .fps: Gets the frame rate of a video
     * .duration: Gets the duration of a video in seconds 
     * .write_videofile(): Saves a video to a new fileSS
